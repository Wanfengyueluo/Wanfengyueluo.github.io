289.生命游戏

:black_nib:解题思路：

  1. 设置额外的状态
    • grid[i][j]==1并且sum<2||sum>3时，设置grid[i][j]=-1表示之前存活，现在死亡
    • grid[i][j]==0并且sum==3时，设置grid[i][j]=2表示之前死亡，现在存活
    • 每次遍历时要考虑grid[i][j]==1和grid[i][j]==-1的情况，因为这两种情况在上个状态是存活的

        public UnionFind(int n) &#123;

            &#125;
        &#125;

        public int find(int p) &#123;

        public void union(int p, int q) &#123;

                return;
            &#125;
            parent[pRoot] = qRoot;

        &#125;
